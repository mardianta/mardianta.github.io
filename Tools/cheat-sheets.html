Ringkasan Perintah Linux Lengkap

Klik nama perintah untuk melihat contoh. Klik ikon i untuk deskripsi, dan klik contoh perintah untuk menyalin.

1. File and Directory Operations Commands

  • ls
    i List files and directories. -l: Long format, -a: include hidden, -h: human-readable.
    ls -l Telah disalin! ls -a Telah disalin! ls -lh Telah disalin!
  • cd
    i Change directory.
    cd /path/to/directory Telah disalin!
  • pwd
    i Print current working directory.
    pwd Telah disalin!
  • mkdir
    i Create a new directory.
    mkdir my_directory Telah disalin!
  • rm
    i Remove files and directories. -r: recursive, -f: force.
    rm file.txt Telah disalin! rm -r my_directory Telah disalin!
  • cp
    i Copy files and directories. -r: recursive.
    cp -r directory destination Telah disalin!
  • mv
    i Move/rename files and directories.
    mv file.txt new_name.txt Telah disalin!
  • touch
    i Create an empty file or update timestamps.
    touch file.txt Telah disalin!
  • cat
    i View the contents of a file.
    cat file.txt Telah disalin!
  • head
    i Display the first few lines of a file.
    head file.txt Telah disalin!
  • tail
    i Display the last few lines of a file.
    tail file.txt Telah disalin!
  • ln
    i Create links between files. -s: symbolic link.
    ln -s source_file link_name Telah disalin!
  • find
    i Search for files and directories. -name: by filename, -type: by file type.
    find /path/to/search -name "*.txt" Telah disalin!

2. File Permission Commands

  • chmod
    i Change file permissions. (u=user, g=group, o=other, +=add, -=remove, =set)
    chmod u+rwx file.txt Telah disalin!
  • chown
    i Change file ownership.
    chown user file.txt Telah disalin!
  • chgrp
    i Change group ownership.
    chgrp group file.txt Telah disalin!
  • umask
    i Set default file permissions.
    umask 022 Telah disalin!

3. File Compression and Archiving

  • tar
    i Create or extract archive files. -c: create, -x: extract, -f: file, -z: gzip.
    tar -czvf archive.tar.gz files/ Telah disalin!
  • gzip
    i Compress files. -d: decompress.
    gzip file.txt Telah disalin!
  • zip
    i Create compressed zip archives. -r: recursively.
    zip archive.zip file1.txt file2.txt Telah disalin!

4. Process Management Commands

  • ps
    i Display running processes. -aux: show all processes.
    ps aux Telah disalin!
  • top
    i Monitor system processes in real-time.
    top Telah disalin!
  • kill
    i Terminate a process. -9: forcefully kill.
    kill PID Telah disalin!
  • pkill
    i Terminate processes based on their name.
    pkill process_name Telah disalin!
  • pgrep
    i List processes based on their name.
    pgrep process_name Telah disalin!
  • grep
    i Search for patterns in files. -i: ignore case, -v: invert match, -r: recursive.
    grep -i "hello" file.txt Telah disalin!

6. Networking Commands

  • ifconfig
    i Display network interface information.
    ifconfig Telah disalin!
  • ping
    i Send ICMP echo requests to a host.
    ping google.com Telah disalin!
  • netstat
    i Display network connections and statistics.
    netstat -tuln Telah disalin!
  • ss
    i Display network socket information.
    ss -tuln Telah disalin!
  • ssh
    i Securely connect to a remote server.
    ssh user@hostname Telah disalin!
  • scp
    i Securely copy files between hosts.
    scp file.txt user@hostname:/path/to/destination Telah disalin!
  • wget
    i Download files from the web.
    wget http://example.com//file.txt Telah disalin!
  • curl
    i Transfer data to or from a server.
    curl http://example.com/ Telah disalin!

7. IO Redirection Commands

  • cmd < file
    i Input of cmd is taken from file.
    grep "text" < file.txt Telah disalin!
  • cmd > file
    i Standard output (stdout) of cmd is redirected to file.
    ls > file_list.txt Telah disalin!
  • cmd 2> file
    i Error output (stderr) of cmd is redirected to file.
    command_that_fails 2> errors.log Telah disalin!
  • cmd 2>&1
    i stderr is redirected to the same place as stdout.
    ls -l > output.log 2>&1 Telah disalin!
  • cmd > /dev/null
    i Discards the stdout of cmd by sending it to the null device.
    rm -rf folder/ > /dev/null 2>&1 Telah disalin!

8. Environment Variable Commands

  • export
    i Sets the value of an environment variable.
    export VARIABLE_NAME=value Telah disalin!
  • echo $VARIABLE
    i Displays the value of an environment variable.
    echo $PATH Telah disalin!
  • env
    i Lists all environment variables.
    env Telah disalin!
  • unset
    i Removes an environment variable.
    unset VARIABLE_NAME Telah disalin!

9. User Management Commands

  • who
    i Show who is currently logged in.
    who Telah disalin!
  • sudo adduser
    i Create a new user account.
    sudo adduser username Telah disalin!
  • sudo userdel
    i Delete a user account. `-r` to remove home directory.
    sudo userdel -r username Telah disalin!
  • su - username
    i Switch to another user account with their environment.
    su - username Telah disalin!
