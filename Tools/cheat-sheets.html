Ringkasan Perintah Linux Lengkap

Klik nama perintah untuk melihat contoh. Klik ikon i untuk deskripsi, dan klik contoh perintah untuk menyalin.

Navigasi File

  • ls
    i Lihat isi direktori. Gunakan `-l` untuk detail, `-a` untuk file tersembunyi.
    ls -l Telah disalin! ls -a Telah disalin!
  • cd
    i Ganti direktori. Gunakan `..` untuk kembali.
    cd /var/www Telah disalin! cd .. Telah disalin!

Manajemen File

  • mkdir
    i Buat direktori baru. Gunakan `-p` untuk membuat direktori induk.
    mkdir proyek-baru Telah disalin! mkdir -p folder/subfolder Telah disalin!
  • rm
    i Hapus file/direktori. Gunakan `-r` untuk folder dan `-f` untuk paksa.
    rm file.txt Telah disalin! rm -rf folder-lama Telah disalin!

Izin & Kepemilikan

  • chmod
    i Ubah izin file. Angka mewakili: (4=r, 2=w, 1=x).
    chmod 777 skrip.sh Telah disalin! chmod 755 skrip.sh Telah disalin! chmod 644 file.txt Telah disalin! chmod -R 777 folder/ Telah disalin!
  • chown
    i Ubah pemilik file. Gunakan `-R` untuk rekursif.
    chown user:group file.txt Telah disalin! chown -R www-data:www-data /var/www Telah disalin!

Teks & Sistem

  • cat
    i Menampilkan isi file.
    cat laporan.md Telah disalin! cat file1.txt file2.txt > gabungan.txt Telah disalin!
  • ps
    i Lihat proses yang sedang berjalan.
    ps aux Telah disalin! ps -ef | grep nginx Telah disalin!

Jaringan & Pengunduhan

  • wget
    i Unduh file dari web.
    wget https://example.com/file.zip Telah disalin!
  • curl
    i Transfer data ke/dari server. `-O` untuk menyimpan file.
    curl -O https://example.com/file.zip Telah disalin!
