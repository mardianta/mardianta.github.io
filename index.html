Standards

In my cybersecurity and penetration testing work, I adhere to several industry standards to ensure the highest quality and effectiveness. These standards include:

  • OWASP (Open Web Application Security Project): A foundation that works to improve the security of software through community-led open-source software projects, local chapters, conferences, and more.
  • NIST (National Institute of Standards and Technology): Provides a cybersecurity framework consisting of standards, guidelines, and best practices to manage cybersecurity-related risk.
  • ISO/IEC 27001: An international standard on how to manage information security, providing requirements for an information security management system (ISMS).
  • MITRE ATT&CK: A globally accessible knowledge base of adversary tactics and techniques based on real-world observations.
